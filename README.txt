================================================================================
PROJECT README
================================================================================

--------------------------------------------------------------------------------
1. DEPENDENCIES & SOFTWARE REQUIREMENTS
--------------------------------------------------------------------------------
To run the notebooks in this repository, ensure you have Python 3.x installed 
along with the following primary libraries:

* TensorFlow / Keras (for model training and inference)
* NumPy (for numerical operations)
* OpenCV (cv2) / Pillow (for image processing)
* Matplotlib (for visualization)
* Jupyter Notebook or VS Code with the Jupyter extension

You can typically install the required packages using pip:
pip install tensorflow numpy opencv-python matplotlib jupyter

--------------------------------------------------------------------------------
2. FILE DESCRIPTIONS
--------------------------------------------------------------------------------

Main Execution Files:
* main.ipynb            : The primary notebook for model training and evaluation. 
                          Running this generates the final saved model.
* predict.ipynb         : The primary inference notebook. Used to load the trained 
                          model and generate predictions on new images.
* model.keras           : The final, trained primary model weights generated by 
                          main.ipynb.

Directories & Configuration:
* Dataset/              : Directory containing the training and validation data.
* test_images/          : Directory containing unseen images for running predictions.
* .vscode/              : Visual Studio Code environment settings.
* PYNB.code-workspace   : VS Code workspace configuration file.

Experimental / Alternative Approaches:
* R-CNN.ipynb           : Experimental notebook testing an R-CNN architecture.
* R-CNN_predict.ipynb   : Inference notebook specifically for the R-CNN model.
* R-CNN.keras           : Saved model weights from the R-CNN experiment.
* local_attention.ipynb : Experimental notebook testing local attention mechanisms.

--------------------------------------------------------------------------------
3. HOW TO RUN THE CODE
--------------------------------------------------------------------------------

Step 1: Setup
Ensure your data is properly extracted and placed inside the "Dataset/" folder, 
and that any images you want to test are placed in the "test_images/" folder.

Step 2: Training (Optional if model.keras already exists)
Open "main.ipynb" in your Jupyter environment. Execute the cells sequentially 
to process the dataset, define the architecture, and train the model. This will 
output/update the "model.keras" file.

Step 3: Inference / Prediction
Open "predict.ipynb". Execute the cells sequentially. This script will load 
"model.keras" and apply it to the images located in the "test_images/" directory 
to generate your final outputs or visualizations.

(Note: If you wish to review the experimental approaches, you can independently 
run R-CNN.ipynb or local_attention.ipynb following a similar workflow.)
